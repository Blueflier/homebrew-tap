
# Blueflier's Homebrew Tap

A personal collection of macOS applications and tools. Currently, this tap is the primary home for **FocusHue**.

## Installation

To add this tap to your local Homebrew installation:

```bash
brew tap Blueflier/tap
```

## Available Casks

### FocusHue

A macOS focus app designed to help you maintain deep work through visual cues and environmental adjustments.

**To install:**

```bash
brew install --cask focushue

```

---

## Maintenance & Troubleshooting

If you have already installed FocusHue and want to update to the latest version:

```bash
brew update
brew upgrade --cask focushue

```

If you encounter issues with the tap, you can reset the connection:

```bash
brew untap Blueflier/tap
brew tap Blueflier/tap

```
